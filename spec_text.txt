1.4 Logo Usage
Minimum size: 120px wide on desktop, 80px on mobile
Clear space: Equal to the height of the "E" in EduMadras on all sides
Dark backgrounds: Use full-color logo (navy + teal)
White navbar: Use full-color logo
NEVER stretch, rotate, recolor, or add effects to the logo
Favicon: Use the book icon portion only, at 32x32px
2. Design Principles
1. Student First — Every decision serves a 17-year-old making a life choice
  Use plain, jargon-free language. "Annual fees" not "tuition quantum"
  Show the most important information first: fees, package, eligibility
  Never bury the counselor contact. It is always one tap away
  Avoid dark patterns — no fake countdown timers, no manipulative CTAs
2. Trust Through Data — Every number must be sourced and dated
  Show "Last updated: Month Year" on all fee/package data
  Display government approval badges (AICTE, UGC, MCI) prominently
  Ratings must show review count alongside the number
  Use official college logo and banner images, not stock photos
3. Speed is a Feature — Sub-2 second loads on 4G mobile
  Lazy-load all images below the fold
  Skeleton loaders on all cards during data fetch
  No layout shift after images load (set explicit width/height)
  Critical CSS inline in <head>. Non-critical deferred
4. SEO is Design — Every page element serves Google indexing
  H1 must contain college name + city for every college page
  FAQ sections on every college page (drives featured snippets)
  Breadcrumb visible on all inner pages
  Structured data JSON-LD injected server-side on every page
5. Mobile First — 73% of Indian students browse on phones
  Design at 375px width first. Expand to desktop as enhancement
  Touch targets minimum 44x44px
  Bottom navigation bar on mobile (not hamburger menu)
  Sticky "Get Free Counseling" CTA on mobile at all times

6. SEO UI Patterns
These are UI elements that directly impact Google search ranking and appearance. Every element here is non-negotiable.
6.1 Page Title Formula (in <title> tag)
  [College Name] — Fees, Courses, Placements [Year] | EduMadras
  Top Engineering Colleges in India [Year] — Rankings, Fees | EduMadras
  Colleges Accepting JEE Advanced [Year] — List, Cutoffs | EduMadras
6.2 Meta Description Formula
  IIT Madras: B.Tech fees ₹8L/yr, avg package ₹21.5L. Rank #1 Engineering. AICTE approved. Compare courses, check eligibility, get free counseling.
6.3 H1 Placement Rule
Every page must have exactly one H1. It must appear in the visible page content above the fold. It must contain the primary keyword for that page.
6.4 FAQ Component (Required on every college page)

8. Accessibility Requirements
EduMadras must meet WCAG 2.1 AA standard. Students with disabilities deserve equal access to education opportunities.
Requirement
Implementation
Color contrast
All text on colored backgrounds must meet 4.5:1 ratio minimum. Navy on white: passes. White on Navy: passes. Check all badge text.
Focus indicators
All interactive elements: 3px solid #1a7a6e outline offset 2px on keyboard focus. Never remove outline with outline:none only.
Alt text
All college logo images: alt="[College Name] Logo". Banners: alt="[College Name] campus". Decorative: alt=""
Semantic HTML
Navigation: <nav>. Main content: <main>. College sections: <section> with aria-label. Sidebar: <aside>. Headings in correct H1>H2>H3 order.
Touch targets
Minimum 44x44px for all interactive elements on mobile. Apply min-h-[44px] to all buttons.
Form labels
All inputs have visible <label> or aria-label. Required fields have aria-required="true". Errors use aria-describedby.
Keyboard navigation
Tab order follows visual reading order. Modal traps focus inside. Escape closes modal/drawer.
Screen reader
Stream badges: aria-label="Engineering college" not just the color. Stats: aria-label="Average placement package: 21.5 lakh rupees".
Loading states
aria-busy="true" on containers during data fetch. aria-live="polite" on result counts when filters change.
Skip link
"Skip to main content" as first focusable element. Visible on focus.
